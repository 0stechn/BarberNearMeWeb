# BarberNearMe — Tanıtım Web Sitesi

BarberNearMe mobil uygulamasının tanıtım (landing) sayfası.

- **Dil:** Türkçe
- **Teknoloji:** Tek dosyalık statik HTML/CSS (`index.html`) — bağımlılık yok
- **Tasarım:** Uygulamayla aynı marka kimliği (sarı `#FFCE38` + siyah `#020000`, makas logo)

## Yerelde açma
`index.html` dosyasına çift tıkla — tarayıcıda açılır, sunucu gerekmez.

## Yayınlama (GitHub Pages)
Repo **Settings → Pages → Source: `main` / root** seçilince
`https://0stechn.github.io/BarberNearMeWeb/` adresinde canlı yayınlanır.
